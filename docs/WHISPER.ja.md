# Whisper 音声認識

本サーバーは、[Whisper.net](https://github.com/terryno/Whisper.net) を介して OpenAI の Whisper モデルを統合し、ローカルで高品質な音声文字起こしを提供します。

## サポートされているモデル

モデルは初回使用時に Hugging Face から自動的にダウンロードされます。

| モデル | サイズ | 特徴 |
|-------|------|----------|
| **Tiny** | 39 MB | 高速、低精度 |
| **Base** | 74 MB | 推奨されるデフォルト |
| **Small** | 244 MB | 高品質 (日本語に最適) |
| **Medium** | 769 MB | ファイル処理用、低速 |
| **Large** | 1550 MB | 最高精度、非常に低速 |

## 特徴

- **ローカル処理**: 初回のモデルダウンロードを除き、音声データがマシン外に出ることはありません。
- **自動言語検出**: 話されている言語を自動的に判別します。
- **翻訳**: 英語以外の音声を直接英語に翻訳して文字起こしできます。
- **システム音のサポート**: YouTube や Web 会議など、Windows 上で再生されている音を文字起こしできます。

## パフォーマンスのヒント

- **CPU 処理**: 現在の実装は Whisper.net の CPU 版を使用しています。
- **日本語の文字起こし**: 日本語の場合、`small` モデルは `base` モデルよりも大幅に精度が向上します。
- **長い音声**: 30秒を超える音声の場合は、`listen` ツールの duration パラメータを長くして直接使用してください。
